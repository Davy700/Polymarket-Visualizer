# Polymarket-Visualizer
This program visualizes the prices of the outcome shares of the current 15 minutes crypto market and the price movement of the adjusted crypto currency in real-time.

# Features of the program
- Automatically accesses the current 15 minutes crypto market based on the calculation of epoch time
- Automatically obtains the "yes" and "no" shares' token ID on the Polygon blockchain
- Subscribes to the current cryptocurrency price using websockets
- Communicates with Polymarket's APIs
- Visualizes the collected data on a plot and updates it
